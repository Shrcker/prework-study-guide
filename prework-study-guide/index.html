HTML

  • The head element contains info about the page.
  • The body element represents the visible content of the page.
  • Sections are semantic groupings for thematic groups in the code.
  • Sections can have classes and id's assigned to them. Classes and id's can be defined in CSS later.
  • Classes and id's can also be manipulated by Javascript elsewhere!

CSS

  • CSS code is defined within curly brackets, { }, within a document.
  • CSS is good for organizing and creating the visual flairs of a page.
  • Classes and id's are usually defined by style sheets near the top of the document, within its own element.

GIT

  • Git is a version control system used for organizing our projects non-linearly and collaborate with other developers.
  • Github is where we store our project repositories in a cloud format. It's essentially what hosts our Git projects.
  • With Git, we can easily track each version of a project and revert to specific states. It's useful for debugging and tracking our progress.
  • Anyone can collaborate with online Github projects.
  • We can also track known issues or bugs with project versions with Github.
  • One can remotely download and access files and repositories onto local work stations.
  • git status: checks what branch we're interacting with.
  • git checkout -b branch-name: creates a new branch and automatically switches to it.
  • git merge [branch]: merges the specified branch's history into the current one.
  • git log: shows all commits in the current branch's history.
  • git fetch [alias]: fetches down all the branches from that Git remote.
  • git push [alia] [branch]: transmits local branch commits to the remote repository branch.
  • git branch: lists your branches.a, * will appear next to the currently active branch.
  • git branch [branch-name]: creates a new branch at the current commit.
  • git rebase [branch]: apply any commits of current branch ahead of a specified one.
  • git reset --hard [commit]: clear staging area, rewrite working tree from specified commit.
